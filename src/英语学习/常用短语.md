# 常用短语


## to some extent

**翻译**：在某种程度上、在一定程度上

**用法**：
- To some extent, I agree with your opinion. （在某种程度上，我同意你的观点。）
- The problem has been solved to some extent. （问题在一定程度上得到了解决。）


## run against

**翻译**：在...背景下运行；针对...运行

**用法**：
- **及物动词**：I ran the test against the fixture. （我针对这个夹具运行了测试。）
- **不及物动词**：The test runs against the fixture. （测试针对夹具运行。）

## there's agreement on

**翻译**：在......上达成一致
**用法**：
- Note that if you are thinking of providing a fix for one of the bugs or feature requests, it's usually
  a good idea to add a comment to the bug to make sure that there's agreement on how we should proceed.
  (请注意，如果你打算修复某个bug或者实现某个功能，通常最好先在该issue下添加一条评论，以确保大家在推进方式上达成一致。)
