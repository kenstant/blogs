## 写作约束

- 第五节（反例与代价）和第六节（回到自己的代码）**不允许只写「TODO」**——这两节是博客与「抄书」的区别，缺了就别发布。
- 代码块用 Java，与原书一致；行数控制在能读完，超长用 `// ... 略` 截断。
- 一句话口诀**用动词开头**，例如「**封装**变化的算法族」「**用工厂代替 new**」，方便回忆。
- 顶部 frontmatter（`title` / `date`）按仓库已有笔记的风格保留。
